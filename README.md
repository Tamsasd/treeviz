# TreeViz

> BST, AVL and TB Tree visualizer, with step-by-step animations and side-by-side comparision of algorithms.

## Features

- **Initial insertions:** Provide a list of numbers to start your trees with
- **Comparision mode:** Compare the selected trees by performing actions simultaneously
- **Fullscreen mode:** Analize the full history of a tree and view the traversals
- **Downloading:** Save the tree as an svg file to your device
- **Autoplay:** Play insert/delete/search sequences automatically, in a choosen speed

## Tech Stack

- **Framework:** React + Vite
- **Language:** TypeScript
- **Frontend:** CSS
